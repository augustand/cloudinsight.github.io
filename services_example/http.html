HTTP

Cloudinsight 监控 http 网站 url 的请求时间，监控其网站可用性以及请求时间等性能状况。

性能指标

Cloudinsight 采集以下 HTTP 性能指标：

  • network.http.response_time

配置 HTTP 监控

网络监控默认在安装 Cloudinsight 探针后会自动生成一些网络流量传输的指标，除此之外想要监控特定地址运行情况，还需要特别配置。

切换路径至 /etc/oneapm-ci-agent。

cd /etc/oneapm-ci-agent

HTTP 连接

开启配置文件 conf.d/http_check.yaml

cp conf.d/http_check.yaml.example  conf.d/http_check.yaml

init_config:
instances:
  - name: My first service
    url: http://dc-cloud.oneapm.com
    timeout: 1

可添加参数：

  • username: 如果测试的页面需要认证需要添加此项
  • password: 如果测试的页面需要认证需要添加此项
  • timeout: 请求超时时间，默认为10s

重启 Agent

重启 OneAPM Cloudinsight Agent，使配置生效。

您也可以通过查看 Agent Info 信息，来验证配置是否成功。当出现以下信息，则代表安装成功。

http_check
----------
  - instance #0 [OK]
  - Collected 1 metric, 0 events & 2 service checks

有关 Agent Info 信息的查看，请访问帮助中心，查看 Cloudinsight Agent 常用操作。
