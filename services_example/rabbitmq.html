RabbitMQ

本章节为 Cloud Insight 支持的平台服务的文档的示例。

分为以下几个部分：

  • 支持的性能指标
  • 如何配置 RabbitMQ 监控
  • 常见的问题

性能指标

Cloud Insight 采集 RabbitMQ 以下性能指标：

指标 具体含义
rabbitmq.node.fd_used 已使用的文件描述符数量
rabbitmq.node.mem_used 内存使用子节数
rabbitmq.node.partitions 网络分区后可见节点的数量
rabbitmq.node.run_queue 等待运行的 Erlang 进程数量
rabbitmq.node.sockets_used 以 socket 方式使用掉的文件描述符数量
rabbitmq.queue.consumer_utilisation 队列消费者采取新消息的时间的比例
rabbitmq.queue.consumers 消费者个数
rabbitmq.queue.memory 与队列关联的 Erlang 进程的内存消耗，包括堆栈，堆和内部结构消耗 /bytes
rabbitmq.queue.messages 队列上的消息队列
rabbitmq.queue.messages.ack.count 传送到客户端并且被确认的消息数量
rabbitmq.queue.messages.ack.rate 每秒传送到客户端并且被确认的消息数量
rabbitmq.queue.messages.deliver.count 消费者接受并且响应的消息数量
rabbitmq.queue.messages.deliver.rate 每秒消费者接受并且响应的消息数量
rabbitmq.queue.messages.deliver_get.count 消费者接受消息（被确认和没有被确认的消息）和从队列当中取出消息（ack 参数传给 basic.get 和 no_ack 参数传给 basic.get） 的总和数
rabbitmq.queue.messages.deliver_get.rate 每秒消费者接受消息（被 ack 和没有被 ack 的消息）和从队列当中取出消息（ack 参数传给 basic.get 和 no_ack 参数传给 basic.get） 的总和数量
rabbitmq.queue.messages.publish.count 发布消息数量
rabbitmq.queue.messages.publish.rate 每秒发布消息数量
rabbitmq.queue.messages.rate 每秒队列中总的消息数量
rabbitmq.queue.messages.redeliver.count deliver_get 的有重新传送标记的消息子集数
rabbitmq.queue.messages.redeliver.rate 每秒 deliver_get 的有重新传送标记的消息子集数
rabbitmq.queue.messages_ready 准备发送给用户的消息数
rabbitmq.queue.messages_ready.rate 每秒准备发送给用户的消息数
rabbitmq.queue.messages_unacknowledged 发送给客户端但至今还没被确认的消息数
rabbitmq.queue.messages_unacknowledged.rate 每秒发送给客户端但至今还没被确认的消息数

配置 RabbitMQ 监控

RabbitMQ Management Plugin

RabbitMQ 的监控需要启用 Management Plugin rabbitmq-plugins enable rabbitmq_management

编辑配置文件

编辑配置文件 conf.d/rabbitmq.yaml，使 Cloud Insight Agent 可以与 RabbitMQ 通信。

init_config:

instances:
  - rabbitmq_api_url: http://localhost:15672/api/
    rabbitmq_user: username # defaults to 'guest'
    rabbitmq_pass: password # defaults to 'guest'

重启 Agent

重启 OneAPM Cloud Insight Agent，使配置生效。

您也可以通过查看 Agent Info 信息，来验证配置是否成功。当出现以下信息，则代表安装成功。

Checks
======

[...]

rabbitmq
--------
  - instance #0 [OK]
    - Collected 8 metrics & 0 events

有关 Agent Info 信息的查看，请访问帮助中心，查看 Cloud Insight Agent 常用操作。
