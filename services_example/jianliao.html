简聊

本章节为 Cloud Insight 支持的平台服务的文档的示例。

Cloud Insight 已经和简聊合作， Cloud Insight 上面的事件流可以通过配置聚合在简聊上面，以下是配置：

1. 前往简聊 • 聚合服务

在简聊中打开聚合服务功能，选择添加第三方服务：Cloud Insight。

点击添加，复制「生成 WebHook 地址」至剪切板。

2. 保存 WebHook 地址

粘贴 WebHook 地址至以下输入框后，点击提交。

此时，在简聊聚合服务中，管理已有服务器项目下已经有 Cloud Insight。

回到简聊主页，在绑定的话题里面就会有从 Cloud Insight 同步过去的事件流。
