Gunicorn

Gunicorn 是一个 Python WSGI UNIX 的 HTTP 服务器。该 Gunicorn 服务器与各种 Web 框架兼容，执行非常简单，轻量级的资源消耗，相当迅速的响应。

Cloudinsight 监控 Gunicorn 请求时间请求数等性能指标。

性能指标

Cloudinsight 采集 Gunicorn 以下性能指标：

指标 具体含义
gunicorn.log.critical 每秒纪录 critical 状态的数量
gunicorn.log.error 每秒纪录 error 的数量
gunicorn.log.exception 每秒纪录 exception 的数量
gunicorn.log.warning 每秒纪录 warning 的数量
gunicorn.request.duration.95percentile 95% 的持续请求时间
gunicorn.request.duration.avg 持续请求平均时间
gunicorn.request.duration.count 每秒收到请求数
gunicorn.request.duration.max 最大持续请求时间
gunicorn.request.duration.median 中位的持续请求时间
gunicorn.request.status.100 每秒产生一个100状态码的响应的请求数
gunicorn.request.status.xxx 每秒产生一个xxx状态码的响应的请求数 （包括所有常见状态码）
gunicorn.requests 每秒收到请求数
gunicorn.workers 仲裁器管理的 worker 数

配置 Gunicorn 监控

1. 保证 Gunicorn 系统中安装有 setproctitle 模块

2. Gunicorn 运行命令模式如下，后面会使用 my_web_app

gunicorn --name my_web_app my_web_app_config.ini

3. 配置 Cloudinsight Agent 连接到 Gunicorn 进程

切换路径至 /etc/oneapm-ci-agent，开启配置文件 conf.d/gunicorn.yaml

cd /etc/oneapm-ci-agent
cp  conf.d/gunicorn.yaml.example   conf.d/gunicorn.yaml

编辑配置文件如下：

init_config:

instances:
  - proc_name: my_web_app

重启 Agent

重启 OneAPM Cloudinsight Agent，使配置生效。

您也可以通过查看 Agent Info 信息，来验证配置是否成功。当出现以下信息，则代表安装成功。

Checks
======

gunicorn
--------
  - instance #0 [OK]
  - Collected 2 metrics, 0 events & 2 service checks

有关 Agent Info 信息的查看，请访问帮助中心，查看 Cloudinsight Agent 常用操作。
