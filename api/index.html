Cloud Insight API 与 SDK

通过Cloud Insight提供的 API 与 SDK 您可以方便的上传任意数据指标，从而实现业务数据的监控和可视化。当前，Cloud Insight 提供两种接入方式，您可以根据您的使用环境，选择适合的方式。

Cloud Insight SDK

点击查看详细说明：SDK

SDK方便您在已经安装了Cloud Insight Agent的主机平台中，采集您需要的指标数据，sdk会自动将数据发送给主机内的Agent，由Agent 统一发送给Cloud Insight的服务器，使用非常的方便，目前SDK提供了以下几种语言的版本。

  • Python
  • Node.js
  • Ruby

Cloud Insight API

API 提供了更加强大的功能，您可以不受任何约束的发送符合Cloud Insight数据格式标准的任意数据，无论本机是否安装了Cloud Insight Agent。目前API正在开发之中。
