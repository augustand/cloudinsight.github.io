图表分享

在进行操作之前请确保安装了Cloudinsight Agent，分享的数据全部来自Agent，开放API 会在2016.5推出，目前可以采用SDK的方式，通过采集自定义的数据，发送给指定的Agent，由Agent 统一上传数据，关于SDK，请参考 API 与 SDK

如何制作分享的图表

  1. 点击侧边栏的仪表盘菜单项
  2. 选择添加一个仪表盘，或者进入平台仪表盘，复制为一个可编辑的副本
  3. 选择某一个Chart的右上角设置按钮，并点击分享
  4. 这里提供了3种分享方式：
    • 直接分享可以查看的URL，您可以拷贝并通过邮件发送给需要查看数据的其他人
    • iframe的版本，您可以将其粘贴至自己的代码中，嵌入到自己的系统中
    • 通过手机扫描二维码，通过手机分享给其他人
