Debian / Ubuntu 安装

Cloudinsight 支持监控 Debian / Ubuntu 系统的磁盘，网络流量，I/O 读写，内存，CPU 使用情况等数据，显示消耗资源较多的进程信息，基本功能完全替代系统自带的监控工具，一键部署轻松查看系统资源消耗情况。效果如下：
