Agent 操作与配置

概述 Cloudinsight Agent 的基本使用方法。如果您尚未安装探针，可以先进行安装。

默认情况下，探针将被安装在 /opt/oneapm-ci-agent 目录。

启动 / 停止 / 卸载

启动探针：

sudo /etc/init.d/oneapm-ci-agent start

停止探针：

sudo /etc/init.d/oneapm-ci-agent stop

重启探针：

sudo /etc/init.d/oneapm-ci-agent restart

删除探针：

sudo yum erase oneapm-ci-agent / apt-get remove oneapm-ci-agent

更新

CentOS / Redhat 更新

sudo yum install oneapm-ci-agent

Ubuntu / Debian 更新

sudo apt-get update && apt-get install oneapm-ci-agent

查看状态

查看探针是否运行：

sudo /etc/init.d/oneapm-ci-agent status

查看探针状态信息：

sudo /etc/init.d/oneapm-ci-agent info

查看配置文件是否正确：

sudo /etc/init.d/oneapm-ci-agent configtest

探针的配置文件：

/etc/oneapm-ci-agent/oneapm-ci-agent.conf
