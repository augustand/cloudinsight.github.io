Windows 安装

Cloudinsight 支持监控 Windows 的磁盘，网络流量，I/O 读写，内存，CPU 使用情况等数据，探针安装成功即完成监控。如果 Windows 上面有 SQL Server，Microsoft Event View，IIS，WMI 等组件，修改相应的配置文件即可监控，效果如下(Wodinws 本身没有系统负载和进程监控的数据)：

选择平台类型

点击添加平台。

选择服务器平台，目前支持 CentOS、Ubuntu、Debian、Windows、Fedora、RedHat、Amazon Linux、Docker、Chef、Puppet 等多种平台，选择 Windows 平台，有2种安装方式：

标准安装

  • 上图中选择适合的版本(32/64)，点击进行下载
  • 下载完成后点击安装

命令行安装

  • 下载 Windows 版 Agent
  • 打开cmd.exe，定位到 Agent 所在目录，执行下面命令：
msiexec /qn /i oneapm-ci-agent.msi CI_LICENSE_KEY="你的 LICENSE_KEY"

当 Cloudinsight 平台列表页面中出现你的服务器名称时，代表着你安装成功：
