Windows 安装

注册

注册 OneAPM，登录账号后点击 CI 导航栏。然后点击添加平台，您也可以根据本文的安装步骤进行安装。

选择平台类型

选择服务器平台，目前支持 CentOS、Ubuntu、Debain、Windows、Fedora、RedHat、Amazon Linux、Docker、Chef、Puppet 等多种平台，选择 Windows 平台，有2种安装方式：

标准安装

  • 上图中选择适合的版本(32/64)，点击进行下载
  • 下载完成后点击安装

命令行安装

  • 下载 Windows 版 Agent
  • 打开cmd.exe，定位到 Agent 所在目录，执行下面命令：
msiexec /qn /i oneapm-ci-agent.msi CI_LICENSE_KEY="你的 LICENSE_KEY"

当 Cloud Insight 平台列表页面中出现你的服务器名称时，代表着你安装成功：
