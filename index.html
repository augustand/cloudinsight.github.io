丰富的平台服务集成

Cloudinsight 集成了数十种互联网流行基础组件的监控，开箱即用，您只需要进行最小化的配置就可以实现复杂的基础组件监控， 免除了传统基础组件监控中的复杂流程。Cloudinsight 采集这些基础组件的权威指标，所有指标都是标准化的格式，您再也不需要 自己定义各种指标，以及思考如何进行各种指标的组合展示，一切就只有两步，安装探针，查看仪表盘，如果您使用的是阿里云、UCloud 等云厂商，甚至不需要安装探针，Cloudinsight 会直接通过这些云厂商提供的 API 收集服务器以及基础组件的性能和健康情况的 指标，并友好的展示在定义好的仪表盘中。

查看如何配置各种基础组件的监控

高度可定制化的仪表盘

丰富的仪表盘展示才能满足您个性化的数据展示需求，Cloudinsight 的仪表盘定制非常的方便，您可以创建任意的仪表盘，并把任意 的指标添加到图表中去，每一个指标曲线都支持丰富的数学运算，无论是基础组件的指标还是您的业务指标都能非常简单、方便的添加到 仪表盘的图表中。

查看如何配置仪表盘

API 与 SDK

您需要上传个性化的业务数据？没问题，Cloudinsight 提供 API 和 SDK，您可以上传任意的指标，并标记上Tag，然后就可以轻松的 在仪表盘上进行各种个性化的数据展示。

查看如何使用 API 和 SDK

全面的告警策略

各种基础组件的监控告警是必不可少的，Cloudinsight 不仅仅支持基础组件的指标告警，您通过 API 和 SDK 上传的任意指标 都可以进行告警监测，这肯定是您期待已久的功能。
